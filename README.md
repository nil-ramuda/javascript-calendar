# javascript_calendar
Macの`cal`コマンドとほぼ同じ機能をJavaScriptで実装
- `node calendar` => 今月のカレンダーを出力
- `node calendar -m 月` => 指定した月のカレンダーを出力
- `node calendar -m 不正な月` => エラーを出力
